## 📌 Overview

**Simple Calculator** is a lightweight web application that allows users to perform basic arithmetic calculations. It features a modern **Soft UI (Neumorphism)** design with interactive hover and click animations, making it visually appealing and easy to use.

The calculator supports:
- Addition, Subtraction, Multiplication, Division
- Percentage Calculation
- Clear Screen (AC) and Backspace (DE)
